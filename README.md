## 📌 About the Project

A fully responsive **e-commerce single product page** for an Apple Watch, built with pure HTML, CSS, and vanilla JavaScript — no frameworks.

---

## ✨ Features

- **Image Gallery** – Thumbnail navigation with smooth fade transition between images
- **Image Zoom** – Hover over the main image to get a zoom-in effect
- **Image Modal** – Expand button opens a full-view modal
- **Color Swatches** – Switch product variant, synced with the gallery
- **Quantity Control** – Increment/decrement with min/max validation
- **Countdown Timer** – Live countdown with hours, minutes, seconds, and frames
- **Add to Cart Toast** – A smooth notification on cart action
- **Wishlist Toggle** – Heart icon toggles filled/outlined with color change
- **Related Products Slider** – Horizontal slider with prev/next arrows, responsive
- **Category Sidebar** – Slide-in sidebar with overlay and close on ESC or backdrop click
- **Back to Top Button** – Appears after scrolling 300px
- **Responsive Design** – Mobile-friendly layout using CSS media queries
